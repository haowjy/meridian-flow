@meridian/agent-edit architecture

The package is a host-agnostic agent editing core. Hosts provide ports for the update journal, live document coordination, document lifecycle, actor sessions, codecs, and the document model.

Write path

createWriteTool() resolves commands, mutates an agent-local Y.Doc, appends durable journal rows, syncs the live projection, and returns a structured outcome.

Thread origins

Forward agent writes still use a stable Symbol per document/thread as the Yjs transaction origin. The origin is only for attribution and same-actor filtering; durable turn grouping comes from journal metadata.

Undo/redo

Reversal is a single cold reconstruction path. It reads checkpoint plus retained updates, selects target mutation rows by turn identity, reconstructs the reversal bytes, persists the reversal/redo record, and syncs the live document.

Compaction

compact() delegates to the journal adapter. It folds old retained updates into a checkpoint and expires reversal records without mutating live documents.
