v1 simplifications

Deliberately deferred — the seams are clean for these to land later without rewrites. Each links to where the deferral is documented in code.

Tool versioning GH #68

No version pinning. Pure resolvers and a stable ResolvedEdit keep the apply layer version-agnostic until a v2 exists.

Durable cold-redo ReversalRecord lookup

Package computes the undo update; the server adapter (Step 9) owns durable storage and retrieval. persistReversal() is the port seam.

View auto-budget / truncation

view returns full content. Thread-level context management is not yet built; expect an auto/outline heuristic later.

Generic concurrent attribution

ConcurrentEditInfo reports human vs agent buckets only. Individual actor names are a server-adapter concern.

Create-on-empty-doc

The DocumentCoordinator port has no create method. create builds content into the local Y.Doc directly.

Multi-document turn reversal

Each document's undo runs independently — no turn-level coordination across documents yet.

Cross-block find — actually supported

Strings containing \n\n are lowered structurally in the resolver and routed to Tier 2 + Tier 3.
